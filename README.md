# Iliham Siraj | Professional Portfolio

Welcome to the source code for my personal professional portfolio! This website serves as a digital resume and project showcase, highlighting my skills in Full-Stack Web Development, Data Analysis, and Project Leadership.

## 🌐 Live Demo
You can view the live version of this portfolio here: **[ilihamsiraj.github.io](https://ilihamsiraj.github.io)** 

## ✨ Features
- **Fully Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Light / Dark Mode Toggle:** Seamless theme switching with system preference detection.
- **Interactive UI Components:** Includes a custom screenshot carousel, interactive experience tabs, and scroll-revealing animations.
- **Glassmorphism Aesthetic:** Utilizes modern, premium CSS design techniques (blurs, transparent layers, dynamic gradients).
- **SEO & Social Ready:** Includes proper OpenGraph meta tags and structured semantics for easy sharing.

## 🛠️ Built With
- **HTML5:** Semantic structuring and accessibility.
- **CSS3:** Custom styling, animations, CSS variables, Grid, and Flexbox (Vanilla CSS, no external frameworks).
- **Vanilla JavaScript (ES6+):** For DOM manipulation, theme management, and interactive UI logic.

## 📂 Repository Structure
- `index.html` — The main HTML document containing all sections.
- `style.css` — All custom styling, animations, and responsive media queries.
- `script.js` — Client-side logic for the dark mode toggle, mobile navigation, tabs, and carousels.
- `assets/` — Contains all images, certificates (PDFs), and document files used in the portfolio.
