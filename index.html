Overview

Edekise Microfinance is my graduation final project, featuring a secure full-stack web application designed to streamline digital banking, client onboarding, and loan management for financial institutions in Ethiopia. The platform includes a responsive user portal for clients alongside a centralized Admin Dashboard for managing branches, tracking real-time audit logs, and overseeing system users. Developed with institutional standards in mind, it incorporates bank-grade 256-bit SSL encryption and protected sessions to ensure robust security and data integrity.
